Release Notes for FP-NET-BLESTAR1 Function Pack

Copyright 2016-2018 STMicroelectronics

 


The FP-NET-BLESTAR1 is a Function Pack for STM32 Cube. The software runs on the STM32 microcontroller and includes drivers for leveraging the functionalities provided by the X-NUCLEO-IDB05A1, a Bluetooth Low Energy STM32 Nucleo expansion board based on the SPBTLE-RF module, and the X-NUCLEO-IDW01M1, a Wi-Fi STM32 Nucleo expansion board based on the SPWF01SA module. Alternatively, the B-L475E-IOT01A Discovery kit for IoT node, equipped with the WiFi ISM43362-M3G-L44 (from Inventek Systems), and the X-NUCLEO-IDB05A1 can be used. The software comes with examples for creating a BLE network and controlling all devices belonging to the network by means of the IBM Watson IoT Cloud services.

Key features are:

  • Complete middleware to build applications based on both BLE and Wi-Fi connectivity allowing to connect STM32 Nucleo boards with both BLE-enabled smartphone/tablet and Cloud based services.

  • Integrated MQTT protocol middleware for easy access to Cloud services provided by IBM Watson IoT.

  • Sample application for NUCLEO-F401RE (connected to both X-NUCLEO-IDB05A1 and X-NUCLEO-IDW01M1 expansion boards) and B-L475E-IOT01A Discovery Kit acting simultaneously as BLE Master and Server.

  • Binary firmware for NUCLEO-F401RE, NUCLEO-L053R8 or NUCLEO-L476RE acting as Peripheral nodes.

  • App for Android/iOS device to control the BLE Star network.

  • Easy portability across different MCU families, thanks to STM32Cube.

  • Free, user-friendly license terms.


The figure below shows the overall architecture.
     
         

  • At the bottom layer there are the HW components: the STM32 MCU, the BlueNRG and the WiFi network processors.

  • The drivers abstract low level details of the hardware and allow the middleware software to provide the Bluetooth LE and the WiFi features in a hardware independent fashion.
  • The included application provides an example of how to use the code.
